# Contributing to Ease.js

Hi! I'm Rudra, the creator of Ease.js. I want to make web development easier for everyone.

### How to help:
1. **Report Bugs**: If something crashes, open an "Issue" on GitHub.
2. **Add Math**: Help me add more formulas to `EasyMath.js`.
3. **Improve Security**: If you find a way to bypass `Security.js`, please let me know!

### Submission Process:
- Fork the repo.
- Create a new branch.
- Send a Pull Request (PR).
- I will review it as soon as I can!
