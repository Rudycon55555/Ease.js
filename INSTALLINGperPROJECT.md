# Local Project Installation

To start a new Ease.js project in a specific folder:

1. Create your project folder: `mkdir my-new-app && cd my-new-app` (if using Bash)
2. Download the installer: `wget https://raw.githubusercontent.com/Rudycon55555/Ease.js/main/EaseJsInstaller.js` (if using wget)
3. Run the installer: `node EaseJsInstaller.js`
4. Verify the folders: You should see `Describe/`, `Routes/`, `Work/`, and the engine files.
